# Backend for Passenges App

Proyecto donde se exponen la API con todas las rutas del backend para la aplicación web de pasajeros del sistema SAICE.

## Rutas
* POST /auth/login
* POST /auth/register

* GET /passenger/edit
* GET /passenger/payments
* GET /passenger/topups
* PUT /passenger/edit

## Tecnologías
* Nodejs 
* Express
* MySQL
